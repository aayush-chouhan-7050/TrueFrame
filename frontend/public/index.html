Key Features

High-Accuracy Detection

Achieves ~99% accuracy on its validation set, trained on a diverse dataset combining FaceForensics++ and Celeb-DF (v2).

Full-Stack Architecture

Built with a Node.js/Express frontend and a Python/Flask backend for a scalable and robust system.

Downloadable PDF Reports

Users can download a professionally formatted PDF summary of the analysis for their records.

How It Works

TrueFrame operates on a two-server model to separate concerns and optimize performance.

1. Frontend Server

Built with Node.js and Express, this server handles video uploads and serves the user interface you're interacting with now.

2. AI Backend Server

A powerful Python and Flask server that hosts our pre-trained PyTorch model. It performs the core analysis by extracting frames and running inference to detect deepfakes.

3. Analysis & Results

The backend aggregates the analysis from video frames to produce a final verdict and confidence score, which is then sent back to you in a clear, easy-to-understand format.

Technology & Performance

Our AI-powered system uses multiple detection methods to provide the most accurate results.

EfficientNet-B0 Model

We use the state-of-the-art EfficientNet-B0 model, known for its high accuracy and computational efficiency, as the foundation of our detection engine.

Robust Training Data

The model was trained on a massive, combined dataset including FaceForensics++ and the challenging Celeb-DF (v2) to recognize a wide variety of deepfake techniques.

Exceptional Performance
